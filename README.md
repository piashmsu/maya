# MYRA — Android AI Voice Assistant

MYRA is a production-ready Android AI Voice Assistant built with Kotlin that uses the
**Google Gemini Live API** (via WebSocket `BidiGenerateContent`) for real-time, voice-to-voice
conversation. It supports phone command execution (calls, SMS, app open/close, system
toggles), Hinglish + English speech understanding, an animated orb UI, prime contacts,
incoming-call announcement & voice control, a floating overlay, and a double-power-press
shortcut.

See `app/` for the Android app module and `docs/` for the full feature spec.
